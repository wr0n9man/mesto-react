# Mesto

### Обзор

Интерактивная галлерея Mesto, сделанная с помощью js-библиотеки React предназначенная для хранения фото из разных уголков мира.

### Функционал

* Добавлять и удалять карточки
* Ставить лайки
* Открывать карточки
* Редактирование профиля
